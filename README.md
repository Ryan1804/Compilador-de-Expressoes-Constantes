Compilador EC1 com analisadores léxico e sintático

Rodar compilador:

```bash
node main.js arquivo.ec1
```

Tokens: Numero, ParenEsq, ParenDir, Soma, Sub, Mult, Div

Erro léxico: `Erro lexico na posicao X: caractere invalido 'Y'`

Erro sintático: `Erro: Esperado ParenDir, encontrado EOF`
